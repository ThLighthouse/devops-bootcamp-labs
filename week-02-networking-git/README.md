# Week 02 - Networking and Git

This folder contains notes and utilities from the Networking, SSH, and Git module of the DevOps Bootcamp.

## Contents

- `notes.md` — learning notes, commands, and key concepts for environment variables, networking, SSH, and Git.
- `network-check.sh` — a script for verifying network connectivity and service availability.
- `app-config-check.sh` — a script for validating simple application configuration settings.

## Topics Covered

- Environment variables and shell configuration
- Network interface and routing commands
- DNS, ping, and port checks
- SSH key-based authentication and remote access
- Git basics: init, clone, commit, push, branch workflow
- Git troubleshooting and workflow practices

## Purpose

Week 2 is focused on practical network troubleshooting, remote access basics, and version control workflows that are essential for DevOps tasks.
