# DevOps Bootcamp Labs

This repository contains hands-on labs, exercises, notes, and study materials from the TechWorld with Nana DevOps Bootcamp.

## Repository Purpose

This repo tracks my progress through DevOps fundamentals and practical exercises, including Linux, Bash scripting, networking, Git, package tools, and database concepts.

## Repository Structure

- `week-01-linux/` — Linux fundamentals, Bash scripting, system information, permissions, and lab exercises.
- `week-02-networking-git/` — Networking, SSH, Git workflows, environment variables, and configuration checks.
- `week-03-db-package-tools/` — Databases, build tools, package managers, and related DevOps concepts.

## Notes

Each week includes `notes.md` files with summaries, commands, and key concepts. Weekly README files describe the folder contents and how to review the work.

## Usage

- Read the `README.md` file in each week folder for module-specific details.
- Use `notes.md` for study summaries and learning observations.
- Review scripts and lab examples in folder subdirectories.

> This repository is a learning portfolio and reference for DevOps practice, not a production project.
