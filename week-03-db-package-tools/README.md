# Week 03 - Databases, Build Tools, and Package Manager Tools

This folder contains notes from the third module of the DevOps Bootcamp, covering database concepts, build tools, and package manager workflows.

## Contents

- `notes.md` — summaries of database types, build tool concepts, package management patterns, and DevOps best practices.

## Topics Covered

- Database fundamentals and common database categories
- Secure configuration and environment-based database settings
- Build tools such as Maven and Gradle
- JavaScript package management with npm
- Artifact creation and deployment workflows
- The role of Docker in build and deployment pipelines

## Purpose

Week 3 is dedicated to understanding how applications are built, packaged, and deployed, as well as how databases fit into the DevOps lifecycle.
