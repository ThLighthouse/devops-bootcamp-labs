# Week 01 - Linux

This folder contains notes, labs, and practice tasks from the Linux module of the DevOps Bootcamp.

## Contents

- `notes.md` — personal notes from the Linux module.
- `system-info.txt` — system information collected from the lab environment.
- `bash-labs/` — Bash scripting practice and automation exercises.
- `nana-demos/` — demo scripts and examples created during the course videos.

## Topics Covered

- Operating system basics
- Virtualization fundamentals
- Linux command line and shell usage
- Files, directories, and the filesystem hierarchy
- Permission and ownership model
- Package management with APT and Snap
- System information and monitoring commands
- Bash scripting fundamentals
