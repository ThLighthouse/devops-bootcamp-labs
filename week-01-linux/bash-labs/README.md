# Bash Labs 

This folder contains Bash scripting practice completed during the Linux module of the DevOps Bootcamp.

## Scripts 

- `hello.sh` - basic script execution
- `hello-user.sh`- command-line arguments and usage handling
- `check-path.sh` - file and directory checks
- `safe-read.sh` - safe file reading with validation
- `disk-check.sh` - disk usage check
- `memory-check.sh` - memory usage check
- `service-check.sh` - systemd service status check
- `system-report.sh` - basic system report
- `mutli-service-check.sh` - checking multiple services using a loop
- `back-dir.sh` - directory backup script
- `cleanup-tmp.sh` - temporary file cleanup practice
- `log-error-check.sh` - log file error checking practice
- `menu-tool.sh` - simple menu-based Bash tool
- `function-practice.sh` - Bash functions practice

## Skills Practiced

- Variables 
- Arguments 
- Exit codes
- Conditions
- File tests
- Loops 
- Functions
- Command substitution
- Basic system commands
- Simple automation tasks
